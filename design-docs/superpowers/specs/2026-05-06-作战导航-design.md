# 作战导航 — 产品设计文档

## 产品背景

作战导航是「销售智能体」的核心功能，属于超级助理（Super Agent）的默认主视图。销售智能体采用 1+N Agent 架构：
- **1 个超级助理**：意图理解、任务分派、上下文管理、成果总结
- **5 个领域专家Agent**：洞察、计划、执行、跟进、教练

作战导航聚合 5 个专家Agent 数据，以指挥舱形态呈现目标和差距，支持差距诊断、路径规划，并反向触发专家Agent 执行行动。

**产品目标**：围绕业务战略重点与经营核心，按角色智能适配差异化作战路径，沉淀标准化执行范式，实现战略逐级拆解、目标全程可控、管理可复制复用、作战达最优效能。

## 服务对象

| 角色 | 视角 | 核心诉求 |
|------|------|----------|
| AR Sales（一线销售） | 个人 | 我的目标-我的差距-我今天该干什么 |
| 战区总经理（管理者） | 团队 | 团队目标-谁掉队了-我该管什么 |

## 交互形态

B（任务流导航）+ C（Agent Feed 流）融合：
- Agent 主动推送今日概览和预警
- 用户按节奏查看目标→差距→路径→行动，有清晰的作战节奏
- 差距诊断和路径建议有明确的操作出口，可一键触发专家Agent

## 平台适配策略

| 维度 | PC 端 | 移动端 |
|------|-------|--------|
| 布局模式 | 多面板指挥舱，信息平铺 | 对话流 + 结构化卡片 |
| Agent 交互 | 侧边 Feed 面板 + 内嵌建议 | 对话气泡主动推送 |
| 操作方式 | 点击按钮/面板 → 触发行动 | 对话指令 + 卡片按钮 + 快捷栏 |
| 信息密度 | 高，一屏看全局 | 中，卡片折叠 + 按需展开 |
| 导航方式 | 固定布局，滚动浏览 | 下滑浏览 + 对话驱动节奏 |

## 信息架构（6 模块）

### 1. 经营目标全景
- **数据来源**：跟进Agent
- **AR Sales**：个人REV/CA目标及达成率、竞争力指标、团队内排名
- **战区总经理**：团队REV/CA、战区排名、各成员达成率排名

### 2. 客户经营情况
- **数据来源**：执行Agent + 跟进Agent
- **内容**：名下客户REV/CA贡献、同比增减、续约/回款状态、流失风险 + 最近拜访、商机推进阶段、方案进展、合同执行状态
- **战区总经理**：团队客户维度聚合 + 异常客户预警

### 3. 标讯看板
- **数据来源**：洞察Agent
- **2种标讯类型**：
  - 意向标讯：高转化潜力，优先跟进，可一键转商机
  - 公开标讯：需及时反馈处理结果（已提报商机 / 无法跟进）
- **过滤规则**：AR Sales 默认按所属战区+行业自动过滤（sales 属于特定战区，行业通常 1 个或多个）
- **排序规则**：高价值置顶 → 即将截止靠前 → 金额降序
- **提醒联动**：即将截止标讯自动推入行动提醒 + Agent Feed；公开标讯超时未反馈标红预警
- **PC 端**：意向/公开双列面板；**移动端**：对话卡片推送

### 4. 战略任务达成
- **数据来源**：计划Agent + 执行Agent
- **内容**：高价值客户经营、海光战核客户经营、重点战役、客户拜访，进度条 + 完成率
- **战区总经理**：团队维度进度 + 落后成员高亮标注

### 5. 差距 & 路径建议
- **数据来源**：AI 综合分析
- **AR Sales**：个人目标差距 → 根因诊断 → 最优达成路径（如"优先跟进客户A → 预计贡献¥200K → 再攻标讯B"）
- **战区总经理**：团队差距 + AI 管理行动建议（如"王五 REV 落后 → 建议1对1绩效面谈"）
- **操作出口**：一键触发专家Agent（生成拜访计划、配置方案、触发模拟陪练等）

### 6. 行动提醒 + Agent 推送
- **数据来源**：多Agent汇总
- **AR Sales**：拜访提醒、重要标讯、任务提醒、会议提醒 + Agent Feed
- **战区总经理**：同上 + 团队管理提醒（下属辅导、过程抽检、绩效面谈、新人带教）

## 作战地图（可视化视图）

在指挥舱中增加地图可视化能力，以中国地图为底图，分层展示销售作战态势。

### 层级结构

三层钻取：**全国 → 大区（若干战区聚合）→ 战区**

| 层级 | 战区总经理 看到 | AR Sales 看到 |
|------|----------------|--------------|
| 🇨🇳 全国 | 各大区达成对比 · 我所属大区排名 · 在全国的位置 | — |
| 🏢 大区 | 各战区对比 · 我管辖战区的排名 · 在大区的位置 | — |
| 📍 战区 | sales/客户/标讯/竞品/渠道 地图分布 | 所属战区 · 我的客户/标讯等 |

每层都有清晰的"你在这里"定位提示。中国区以上（管理驾驶舱）不在本次范围。

### 地图要素（战区层）

| 要素 | 视觉 | 交互 |
|------|------|------|
| Sales | 蓝色圆点，标注姓名+达成率 | 点击进入该sales个人作战导航 |
| 客户 | 绿色标签，异常客户标红 | 点击进入客户经营详情 |
| 标讯 | 橙色标签，高价值加星标 | 点击跳转标讯看板详情 |
| 竞品 | 紫色标签 | 点击查看竞品动态 |
| 渠道伙伴 | 蓝色标签 | 点击查看伙伴详情 |

### 地图交互

- **缩放/平移**：缩小汇聚到上级视图，放大进入下级视图
- **点击图钉**：弹出详情卡片（关键指标 + 最近动态）→ 进一步跳转完整视图
- **图层筛选**：按类型显示/隐藏（只看客户 / 只看标讯 / 只看竞品 / 全部）
- **汇总面板**：地图侧边显示当前区域统计（sales数、客户数、标讯数、竞品数）
- **跨视图联动**：地图点击标讯 → 标讯看板；点击客户 → 客户经营详情
- **面包屑导航**：全国 > 华东大区 > 浙江战区，支持点击任意层级跳转

## 核心交互流程

### 流程一：AR Sales 每日启动
打开超级助理 → 作战导航（默认视图）→ Agent 主动推送今日概览 → 查看目标全景 → 聚焦差距 & 路径 → 处理今日行动 → 采纳建议/触发行动

### 流程二：差距深挖 → 触发作战行动（核心闭环）
差距告警 → AI 诊断根因 → 路径建议呈现 → 一键调用专家Agent（如：作战计划Agent 生成拜访计划，洞察Agent 获取客户资料）→ 行动确认执行

### 流程三：战区总经理管理干预
团队预警（成员排名异常）→ 一键下钻到该成员个人作战导航 → AI 管理建议 → 执行管理动作（添加面谈日程、抽检拜访记录、分配带教等）→ 同步到行动提醒

### 流程四：教练闭环
- **优秀案例沉淀**：执行结果 → AI 自动判定优秀 → 作战导航展示 → 教练Agent 结构化沉淀入库 → 复用于同类场景/新人培训
- **问题辅导**：执行遇阻/指标落后 → 教练Agent 诊断 → 分级响应（轻度→建议卡片 / 中度→知识问答/培训 / 重度→1对1模拟陪练）→ 作战导航呈现 → 效果追踪

## Agent 数据/触发关系

| Agent | 作战导航拉取 | 作战导航触发 | 主动推送到作战导航 |
|-------|-------------|-------------|-------------------|
| 洞察 | 标讯列表、客户360画像 | 获取拜访资料包 | 重要标讯到期、高价值标讯通知 |
| 计划 | 拜访计划、外呼计划、任务分派 | 根据差距生成/调整计划 | 计划到期提醒 |
| 执行 | 拜访记录、商机数据、合同数据 | 生成方案/报价/标书 | 商机阶段变更、拜访完成通知 |
| 跟进 | BI看板、目标达成、预警数据 | 创建自定义监控规则 | 指标阈值预警、达成率异常 |
| 教练 | 知识库、优秀案例、培训内容 | 触发辅导/问答/模拟陪练、沉淀优秀案例 | 优秀案例评选、辅导建议推送 |

## 导航隐喻设计

- **目标路径可视化**：差距 → 根因 → 路径选项 → 预期贡献 → 行动触发
- **节奏监控**：Q2 已过X%，按节奏应达成Y%，当前达成Z%，前方"拥堵/畅通"
