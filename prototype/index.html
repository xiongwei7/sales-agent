🎯 作战导航 · 能力介绍

定位：作战导航是超级助理的核心功能视图（非独立 Agent），是销售智能体的默认主界面。它聚合 5 个专家 Agent 的数据，以指挥舱形态呈现目标和差距，支持差距诊断与路径规划，并反向触发专家 Agent 执行行动。

6 大信息模块

模块数据来源AR Sales战区总经理
📊 经营目标全景跟进Agent个人 REV/CA/竞争力指标 + 排名团队 REV/CA + 战区排名 + 成员排名
👤 客户经营情况执行 + 跟进名下客户贡献/回款/风险/动态团队客户聚合 + 异常预警
📡 标讯看板洞察Agent按战区+行业过滤，意向/公开标签区分团队标讯分配 + 高价值跟踪
🎯 战略任务达成计划 + 执行四类任务进度条团队进度 + 落后成员标注
🔍 差距 & 路径建议AI 综合分析个人差距 + 最优达成路径 + 一键触发团队差距 + 管理行动建议
📋 行动提醒 + Feed多 Agent 汇总拜访/标讯/任务/会议 + Agent 推送同上 + 团队管理提醒

核心能力

🔄

🧠
智能诊断

行动调度
反向触发专家 Agent：生成拜访计划、获取客户资料、触发模拟陪练等

作战地图

👥
角色适配

📱
