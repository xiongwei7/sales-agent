🎯 作战导航 · 功能定位

作战导航是超级助理的核心功能视图，非独立 Agent。它是销售智能体的默认主界面，聚合 5 个专家 Agent 的数据，以指挥舱形态呈现目标和差距，支持差距诊断与路径规划，并反向触发专家 Agent 执行行动。

💡 核心设计原则

🔄
聚合而非替代
作战导航不自己产生数据，从各专家 Agent 拉取数据做整合展示，是「驾驶舱」而非「发动机」
🚀
展示 + 调度
既能聚合展示各 Agent 输出，又能反向触发行动 —— 发现差距后一键调用计划 Agent 生成拜访计划
👥
角色智能适配
同一框架，不同内容。AR Sales 看「我」，战区总经理看「团队」。按角色差异化呈现作战路径
📱
双端形态适配
PC 指挥舱多面板高密度 · Mobile Agent 对话 + 卡片交互。功能对等，形态随端适配
🗺️
路径 + 节奏
既有目标路径可视化（差距→根因→路径→触发），又有节奏监控（Q2已过X%，应达Y%，当前Z%）
📋
沉淀可复用
标准化执行范式沉淀，优秀案例自动入库，管理经验可复制，实现作战效能最大化
